植物数据结构（共70+字段）
一、基础信息
字段	类型	必填	说明	示例
common_name	字符串	是	中文俗名	龟背竹
aliases	字符串	否	别名，用顿号分隔	蓬莱蕉、电线兰
scientific_name	字符串	是	拉丁学名	Monstera deliciosa
slug	字符串	是	英文短横线，唯一	monstera-deliciosa
family	字符串	否	中文科名	天南星科
genus_name	字符串	否	中文属名	龟背竹属
plant_type	字符串	否	植株品类	草本 / 木本 / 藤本 / 肉质
growth_cycle	字符串	否	生长周期	一年生 / 二年生 / 多年生
ornamental_type	字符串	否	观赏分类	观叶 / 观花 / 观果 / 多肉 / 水培
origin	字符串	否	原产地	中美洲
description	长文本	否	详细描述（50-100字）	
short_description	短文本	否	简短描述（20字以内）
